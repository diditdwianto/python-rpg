# ASCII Knight - Cookie RPG

A browser-based RPG with ASCII art character and cookie currency system.

## Setup

```bash
cd py
pip install -r requirements.txt
python server.py
```

Then open `static/index.html` in your browser.

## How to Play

- **Cookies** generate automatically (+1/second)
- Use cookies to buy equipment from the shop
- Equip items to increase your stats (HP, MP, Damage, Defense)
- The knight wanders around automatically on the game screen

## Controls

- **Shop**: Buy equipment with cookies
- **Inventory**: View and equip items
- **Quests**: View available quests
- **Save**: Save your progress to the server
